# 🚀 Quick Start Guide (Phone Only)

اگر آپ کے پاس صرف موبائل فون ہے:

## ১. اگر آپ ڈیپلائے کرنا چاہتے ہیں:

### Railway پر Backend ڈیپلائے کریں:
1. https://railway.app پر جائیں
2. GitHub سے login کریں
3. اپنا Repository select کریں
4. `backend` folder select کریں
5. Environment variables شامل کریں
6. Deploy بٹن دبائیں

### Vercel پر Frontend ڈیپلائے کریں:
1. https://vercel.com پر جائیں
2. GitHub سے login کریں
3. اپنا Repository import کریں
4. `frontend` folder select کریں
5. Deploy بٹن دبائیں

### لنک استعمال کریں:
- Vercel سے ملنے والا URL اپنے فون میں کھولیں
- ہو گیا! 🎉

---

## ٢. API انٹیگریشن:

### Remove.bg API Key حاصل کریں (مفت):
1. https://remove.bg/api پر جائیں
2. Sign up کریں (مفت)
3. API Key کاپی کریں
4. Railway environment میں شامل کریں

---

## ٣. اگر کوئی مسئلہ ہو:

### Upload fail ہو رہی ہے:
- Backend URL درست ہے یقینی بنائیں
- Vercel Dashboard میں environment variable چیک کریں

### PDF generate نہیں ہو رہی:
- Backend میں Sharp install ہے یقینی بنائیں
- `npm install sharp` چلائیں

### Background remove نہیں ہو رہی:
- Remove.bg API key شامل کریں
- API key صحیح ہے یقینی بنائیں

---

✅ 📱 موبائل پر فوٹو لیں یا اپ لوڈ کریں

✅ 👔 کپڑوں کا اسٹائل منتخب کریں
